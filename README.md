# AI 编程学习指南 —— 使用 Claude Code

## 什么是 AI 编程？

AI 编程是指利用人工智能助手（如 Claude Code）来辅助软件开发过程，包括但不限于：

- **代码生成**：用自然语言描述需求，AI 自动生成代码
- **代码审查**：让 AI 帮你检查代码中的 Bug 和优化点
- **重构与优化**：AI 帮你改进代码结构和性能
- **文档生成**：自动生成注释、README、API 文档等
- **测试编写**：AI 帮你生成单元测试、集成测试
- **调试辅助**：描述问题，AI 帮你定位和修复

## 为什么选择 Claude Code？

Claude Code 是 Anthropic 推出的命令行 AI 编程助手，它的优势包括：

1. **深度理解代码库**：可以读取整个项目，理解上下文
2. **直接操作文件**：可以读取、编辑、创建文件
3. **执行命令**：可以运行 shell 命令、安装依赖、执行测试
4. **多文件协调**：能同时修改多个文件，保持代码一致性
5. **交互式对话**：可以持续对话，逐步完善需求
