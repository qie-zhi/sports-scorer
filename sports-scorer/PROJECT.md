# 🏆 Sports Scorer —— AI 编程学习项目 #1

## 项目概述

**Sports Scorer** 是一个用 React Native 开发的安卓计分应用，支持排球、篮球、足球等运动的实时比分记录。

这是我学习 Claude Code AI 编程的第一个实战项目。

---

## 为什么选这个项目？

| 特点 | 说明 |
|------|------|
| 功能清晰 | 需求一目了然，不会有歧义 |
| 难度适中 | 适合初学，但包含足够的技术深度 |
| 实用性强 | 真实可用，打球时真的能用上 |
| 扩展空间大 | 从简单计分到复杂功能，逐步迭代 |
| 离线可用 | 无需服务器，本地运行 |

---

## 开发方式选择

经过对比四种方案后，选择 **React Native**：

- ⭐ **与 Claude Code 协作效率最高**（JS/TS 是 Claude 最擅长的领域）
- ⭐ **一份代码，Android + iOS 双端运行**
- ⭐ **热重载，改代码立即可见效果**
- ⭐ **社区成熟，生态丰富**

> 详细对比见 [TECH-STACK.md](./TECH-STACK.md)

---

## 第一版功能

- ✅ 两队独立加减分
- ✅ 自动局分管理和换局
- ✅ 排球 25 分制（可切换篮球/足球）
- ✅ 撤销误操作
- ✅ 数据本地持久化（杀进程不丢）
- ✅ JSON 文件导出分享
